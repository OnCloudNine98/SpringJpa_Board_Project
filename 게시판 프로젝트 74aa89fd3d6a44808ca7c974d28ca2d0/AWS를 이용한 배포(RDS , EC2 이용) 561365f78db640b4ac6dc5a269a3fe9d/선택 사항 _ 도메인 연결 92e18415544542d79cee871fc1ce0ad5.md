# 선택 사항 _   도메인 연결

Port Forwarding 처리 하고 나니 매번 IP 주소를 통해 직접 접속 하기에는 불편함이 있다.
따라서 도메인 이름을 따로 설정한 후 IP 와 연결 하여 접근을 간단히 하는 과정이다.

<aside>
💡 **DNS(Domain Name System)**

</aside>

### **1. 특정 사이트를 통해 도메인을 구매**

도메인 관련 사이트(ex. 가비아) 를 통해 도메인을 구매한다.
